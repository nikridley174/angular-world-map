Angular World Map Explorer

Interactive Angular application that displays a world map using SVG and retrieves country data from the World Bank API.

Users can click on a country to view details such as capital, region, income level, and additional metrics.

🔗 Live Demo

https://nikridley174.github.io/angular-world-map/

🚀 Features
- Interactive SVG world map with click events
- Real-time country data via World Bank API
- Angular routing with default redirect
- Responsive two-column layout
- Modular Angular service architecture

🛠 Tech Stack
- Angular 20
- TypeScript
- REST APIs (World Bank)
- SVG interaction
- GitHub Pages deployment

⚙️ Setup
- git clone https://github.com/nikridley174/angular-world-map.git
- cd angular-world-map
- npm install
- ng serve

📦 Environment
- Angular CLI: 20.x
- Node: 22.x
- npm: 11.x
