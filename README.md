⭐ Angular World Map Explorer

Interactive world map built with Angular that displays real-time country data using the World Bank API. Users can click on any country to view details such as capital, region, and income level.

🔗 Live Demo

https://nikridley174.github.io/angular-world-map/

🚀 Features
Interactive SVG world map with click events
Country data fetched dynamically via REST API
Displays capital, region, income level, and additional attributes
Angular routing with default redirect to map view
Responsive two-column layout (map + data panel)
🛠 Tech Stack
Angular 20
TypeScript
HTML / CSS
World Bank API
SVG-based UI
GitHub Pages (deployment)
⚙️ Setup Instructions
git clone https://github.com/nikridley174/angular-world-map.git
cd angular-world-map
npm install
ng serve
📦 Angular / Environment
Angular CLI: 20.1.x
Node: 22.x
npm: 11.x
